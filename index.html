Music Hits Analysis Dashboard

Radar Chart

Compare musical characteristics between hit and non-hit songs using a radar chart visualization. This view provides an intuitive overview of how different musical features contribute to a song's success.

Parallel Coordinates

Explore relationships between musical characteristics using parallel coordinates. This visualization allows you to see patterns and correlations between different features, with individual lines for each song and highlighted averages for hits and non-hits.

Chord Diagram

Discover correlations between musical characteristics and technical features, visualizing the strength of relationships through an interactive chord diagram.

Cross-Platform Matrix

Compare song performance across different platforms using an interactive scatter plot matrix. Identify patterns in cross-platform success through pairwise comparisons.

Platform Performance Analysis

Analyze streaming metrics across different platforms with an interactive parallel coordinates visualization. Filter by genre and hit status to explore performance patterns across Spotify, YouTube, TikTok, and more.

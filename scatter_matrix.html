Cross-Platform Performance Analysis

Understanding Cross-Platform Success

This scatter plot matrix shows how songs perform across different streaming platforms. Each dot represents a song, and its position shows how well it performs on two platforms.

How to Read the Matrix

  • Grid Layout: The visualization is a 5x5 grid where each cell compares two platforms
    • Platform names appear in the diagonal cells
    • Each off-diagonal cell is a scatter plot comparing two platforms
  • Reading a Cell:
    • X-axis (horizontal): Performance on the platform shown at the bottom
    • Y-axis (vertical): Performance on the platform shown on the left
    • Both axes use logarithmic scales to handle the wide range of values
  • Interpreting Patterns:
    • Diagonal trend (bottom-left to top-right): Strong correlation between platforms
    • Scattered points: Weak correlation between platforms
    • Clusters: Groups of songs with similar performance patterns
  • Interactive Features:
    • Hover over any point to see the song details and exact values
    • Use the genre selector to focus on specific music genres

Example Interpretation

If you see points forming a diagonal line in a cell (bottom-left to top-right), it suggests that songs popular on one platform tend to be popular on the other. Scattered points indicate that success on one platform doesn't necessarily translate to success on another.

Numbers are formatted in K (thousands), M (millions), or B (billions) for readability.
